格网计算说明 格网配置用于定义参与降雨差分计算的设备样本和计算约束。启用后，系统会按计划输出 1、12、24 小时格网降雨 Excel 数据。

基础信息

定义格网名称、空间分辨率和启用状态，便于运行监控和输出文件识别。
建议包含区域和分辨率信息，方便在列表、工作台和导出文件中快速识别。
用于描述格网尺度，建议与实际中心点间距或业务输出精度保持一致。
只有启用的格网会参与定时降雨差分计算和 Excel 输出。

设备样本

维护参与该格网计算的设备号集合，设备坐标和预测数据会作为差分输入。
支持英文逗号、空格或换行分隔。设备号需与设备管理中的 sncode 一致，否则不会匹配到有效心跳和预测数据。

计算约束

控制格网中心点计算时的最少样本数量和空间距离约束。
当有效设备数低于该值时，计算结果可信度不足，建议保持默认 3 台以上。
用于过滤距离过近的样本点，降低单点密集分布对差分结果的影响。
